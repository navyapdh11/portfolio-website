# Google Business Profile (GBP) Optimization Checklist

## 1. Core Profile Setup
- [ ] Business Name matches registered name exactly (no keyword stuffing).
- [ ] Primary Category: "End of Lease Cleaning Service" / "House Cleaning Service".
- [ ] Phone: Local number verified in "+61 2 XXXX XXXX" format.
- [ ] Website: Includes UTM parameters (`?utm_source=google&utm_medium=organic`).
- [ ] Hours: Includes all public holidays (Australia Day, ANZAC Day, etc.).

## 2. Content & Trust Signals
- [ ] Description (750 chars): Focus on "bond-back guarantee", "WHS compliance", and "GST inclusive".
- [ ] Services: List all services with pricing: "From $XX AUD".
- [ ] Products: Upload 4+ cleaning packages with pricing and images.
- [ ] Attributes: Select all that apply (e.g., "Women-led", "Eco-friendly", "WHS Compliant").

## 3. Media Strategy (Critical)
- [ ] Logo: High-res square (400x400px min).
- [ ] Cover Image: High-res (1080x608px) featuring professional team.
- [ ] Interior/Work Photos: Minimum 10 images showing cleaning in progress (uniformed staff).
- [ ] Video: 30s team introduction or service demo.

## 4. Review & Engagement Loop
- [ ] Response Protocol: 24h turn-around for all reviews.
- [ ] Solicitation: SMS/Email sent 2 hours post-service.
- [ ] Compliance: Clear ACCC/ACL disclosure (No incentives for reviews).
- [ ] Q&A Seeding: Populate with local FAQs for each suburb/state.

## 5. Analytics & Monitoring
- [ ] Tracking: Monitor search queries, direction requests, calls, and clicks.
- [ ] Sentiment analysis: Monitor keywords "bond", "punctual", "professional".
- [ ] Competitive Monitoring: Audit top 3 local competitors in every metro region weekly.
